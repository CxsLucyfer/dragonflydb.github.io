Developer Advocate

Location: Remote

Dragonfly is a modern replacement for Redis and Memcached and probably the fastest in-memory store in the universe. It implements novel algorithms and data structures on top of a multi-threaded, shared-nothing architecture. As a result, Dragonfly is 30% more memory efficient and x25 more performant compared to Redis even on a single device. It’s extremely simple to set up, maintain, scale and support millions of QPS and hundreds of GBs of memory workloads.

Dragonfly is a disruptive project with a new technological roadmap. It requires an experienced developer advocate to communicate its innovative approach for memory datastores. A successful developer advocate will inspire and equip community developers to build the next generation of amazing real time applications on top of Dragonfly.
